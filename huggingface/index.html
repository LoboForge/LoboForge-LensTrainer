LoboForge LensTrainer

Config-driven LoRA trainer for microsoft/Lens-Base. Flow-match training, ComfyUI export, 24GB presets.

GitHub: LoboForge/LoboForge-LensTrainer

Quick install

git clone https://github.com/LoboForge/LoboForge-LensTrainer.git
cd LoboForge-LensTrainer
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
bash scripts/install_microsoft_lens.sh
export PYTHONPATH="$(pwd)/vendor/Lens:$PYTHONPATH"
huggingface-cli login

Train

python train.py configs/train_lora_lens_base_24gb.yaml \
  --dataset-path /path/to/images \
  --output-dir ./output/my-lora \
  --model-repo microsoft/Lens-Base \
  --steps 4000 --disable-mxfp4
